Lit GIPHY

Anything! Any user input that is submitted on the homepage returns any relevant content from the GIPHY database of GIF images.

This simple web app allows users to enter and submit a keyword on the homepage. The page then sends a request to the Giphy API based on that keyword, receives and parses the response, and displays relevant images on the page.

API stands for Application Programming Interface. An API is essentially a software intermediary that allows two applications to communicate with each other. For example, third party travel sites, such as Expedia or Kayak, interact with airline APIs in order to populate data to the user in the form of flights and hotel bookings.
