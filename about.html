About this project

This project serves as a basic search engine web app. The main page consists of a title banner as well as a text input field with which the user can search for GIF images of their choosing. The search function is executed with a button adjacent to the search input field. Once the user inputs a key word and executes the search, a series of relevant GIF images are displayed on the page. The web app makes use of the GIPHY API in order to return data to the page in the form of GIF images. The technologies used in this project are HTML, CSS, Bootstrap, JavaScript, and jQuery.
